# Задача 1: Установка

## Описание решения домашнего задания

В данной задаче я решил применить некоторые ранее полученные теоретические знания нетологии на практике, чтобы их закрепить.

Например: 
- первое, что хотелось сделать, так это добавить в отдельные коллекции (ArrayList) директории и файлы. Мне показалось это наиболее "захардкоженным" вариантом, чем Enum.
- Использовал набор именованных констант, так как хотел применить его на практике, чего не делал ранее.
- Прошу меня извинить, но был вынужден отказаться от предложенного варианта использования StringBuilder для накопления строк логов работы программы. В качестве реализации логирования, использовал возможности класса FileWriter, обернутый в BufferedWriter. Отказался от StringBuilder, так как представил, что он может разрастись до outOfMemory при возможных больших объемах данных.
